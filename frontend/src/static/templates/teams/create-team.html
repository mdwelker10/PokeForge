User Preferences:

Only valid teams will be accepted. Here are the guidelines for making a valid team:
  • Between 1 and 6 Pokemon are specified
  • Every specified Pokemon has an ability that it is allowed to have
  • Every specified Pokemon has between 1 and 4 moves it is allowed to have
  • Not all Pokemon need a held item, but if it has one it must be one with a use in battle
Note: Currently the data is only up to Generation 8
To view a Pokemon to see moves and abilities it can have, you can search for it over at the Pokemon page or click its sprite when it appears next to the "Pokemon Name" input box (Note: the sprite might not appear on smaller screens).
